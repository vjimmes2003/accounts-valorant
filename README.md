# Pollo Valorant Ranks

App local para consultar rangos de Valorant para la lista de cuentas cargada.

## Cómo usar

1. Abre `server.js` con Node.js 18 o superior.
2. Entra en `http://localhost:3000`.
3. Pega tu API key de HenrikDev en el campo correspondiente.
4. Pulsa `Actualizar rangos`.
5. En Windows también puedes abrir `start-server.bat`.

## Notas

- La app guarda la API key y la caché solo en tu navegador.
- El acceso a rango depende de HenrikDev, que a su vez consulta Riot.
- Los botones de Tracker siguen estando disponibles por cuenta.
